Supported Games

MAME4droid (Current) uses game ROMs from the current desktop MAME. Note: This is different than the ROMs used in MAME4droid (0.139u1). These ROMs cannot be used interchangeably.

ROM Location and Naming Convention

ROMs go in the "roms" folder of your MAME4droid installation. By default (Internal storage, chosen on first run) that's the app's own private folder, normally /storage/emulated/0/Android/data/com.seleuco.mame4d2024/files/roms, and you copy your ROMs there from a PC using the Android Debug Bridge (ADB) tool.

Alternatively, you can pick External Storage at first run (or later, see below) and choose any folder on your device instead, using Android's own folder picker. Reading from there is a little slower, but nothing gets deleted when you uninstall the app, and you can copy ROMs in/out with any regular Android file manager, no PC needed. See the Installation section for the full details on both options.

Additionally, you can add other selected resources like artwork/samples by creating artwork/samples/software subfolders yourself (see Installation for exactly where). If you are an advanced user, you can also edit the .ini files in the main installation folder to fully customize every path.

You are not locked into your first choice: ROMs may be relocated anytime under Options->Settings->General->Change ROMs Path.

Note: File and directory names are case-sensitive. MRDO.ZIP is not the same as mrdo.zip. Always use ZIPped ROMs with filenames in lower case.
